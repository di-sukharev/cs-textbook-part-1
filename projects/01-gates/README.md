## HDL (—è–∑—ã–∫ –æ–ø–∏—Å–∞–Ω–∏—è –∂–µ–ª–µ–∑–∞ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞)

–í –Ω–∞—á–∞–ª–µ 70-—Ö –∫–æ–º–ø—å—é—Ç–µ—Ä—ã –ø–∞—è–ª–∏ —Ä—É–∫–∞–º–∏. –ß—Ç–æ–±—ã —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å —á–∏–ø, –∏–Ω–∂–µ–Ω–µ—Ä –ø–µ—Ä–µ–≤–æ–¥–∏–ª –µ–≥–æ –∏–∑ —Ç–∞–±–ª–∏—Ü—ã –∏—Å—Ç–∏–Ω–Ω–æ—Å—Ç–∏ –≤ –ª–æ–≥–∏—á–µ—Å–∫–∏–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ—Ç–æ–º —É–ø—Ä–æ—â–∞–ª –∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª –∫–∞–∫ —á–µ—Ä—Ç–µ–∂ —ç–ª–µ–∫—Ç—Ä–æ—Å—Ö–µ–º—ã.

–í XXI –≤–µ–∫–µ –∫–æ–º–ø—å—é—Ç–µ—Ä–Ω—ã–µ —Å–∏—Å—Ç–µ–º—ã –ø—Ä–æ–µ–∫—Ç–∏—Ä—É—é—Ç –≤ —Å–∏–º—É–ª—è—Ç–æ—Ä–∞—Ö –∂–µ–ª–µ–∑–∞, –∫–æ—Ç–æ—Ä—ã–µ –≤—Å–µ –¥–µ–ª–∞—é—Ç –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏. –ò–Ω–∂–µ–Ω–µ—Ä, –≤ –ø—Ä–æ–≥—Ä–∞–º–º–µ, –ø—Ä–æ–µ–∫—Ç–∏—Ä—É–µ—Ç —á–∏–ø –Ω–∞ —è–∑—ã–∫–µ HDL, –∞ –ø–æ—Ç–æ–º HDL-—Å—Ö–µ–º–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –º–∞—Å—Å–æ–≤–æ–π –ø–µ—á–∞—Ç–∏ —á–∏–ø–æ–≤ —Ä–æ–±–æ—Ç–∞–º–∏ –Ω–∞ –∑–∞–≤–æ–¥–µ.

–ß—Ç–æ–±—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã–µ —Å—Ä–µ–¥—Å—Ç–≤–∞ –ø—Ä–æ–µ–∫—Ç–∏—Ä–æ–≤–∞–Ω–∏—è —á–∏–ø–æ–≤ ‚Äî –Ω—É–∂–Ω–æ –≤–ª–∞–¥–µ—Ç—å —è–∑—ã–∫–æ–º –æ–ø–∏—Å–∞–Ω–∏—è –∂–µ–ª–µ–∑–∞ HDL.

–í —ç—Ç–æ–º –ø—Ä–æ–µ–∫—Ç–µ –º—ã —Å–æ–±–µ—Ä–µ–º —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã–µ —Å–æ—Å—Ç–∞–≤–Ω—ã–µ –≥–µ–π—Ç—ã –∏ –ø—Ä–æ—Å—Ç–µ–π—à–∏–µ —á–∏–ø—ã –≤ —Å–∏–º—É–ª—è—Ç–æ—Ä–µ –∂–µ–ª–µ–∑–∞, –∏—Å–ø–æ–ª—å–∑—É—è HDL.

## –°–∏–Ω—Ç–∞–∫—Å–∏—Å HDL

HDL (hardware description language) ‚Äî —ç—Ç–æ —è–∑—ã–∫ –æ–ø–∏—Å–∞–Ω–∏—è –∞–ø–ø–∞—Ä–∞—Ç–Ω–æ–≥–æ –æ–±–µ—Å–ø–µ—á–µ–Ω–∏—è (–∂–µ–ª–µ–∑–∞). HDL —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –≤ –∫–æ–Ω—Ç–µ–∫—Å—Ç–µ –ø—Ä–æ–≥—Ä–∞–º–º—ã-—Å–∏–º—É–ª—è—Ç–æ—Ä–∞ –∂–µ–ª–µ–∑–∞. –ü—Ä–æ–µ–∫—Ç–∏—Ä–æ–≤–∞—Ç—å —á–∏–ø –≤ —Å–∏–º—É–ª—è—Ç–æ—Ä–µ –∂–µ–ª–µ–∑–∞ –ø—Ä–æ—â–µ, —á–µ–º –ø–µ—Ä–µ–ø–∞–∏–≤–∞—Ç—å –µ–≥–æ –≤–∂–∏–≤—É—é. –¶–µ–Ω–∞ –æ—à–∏–±–∫–∏ –º–µ–Ω—å—à–µ, –ø—Ä–æ—Ü–µ—Å—Å —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–∏ –±—ã—Å—Ç—Ä–µ–µ.

–õ—é–±–æ–π —è–∑—ã–∫ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ —Ç–æ–∫–µ–Ω–æ–≤, –≤ —Ç–æ–º —á–∏—Å–ª–µ HDL. –¢–æ–∫–µ–Ω—ã ‚Äî —ç—Ç–æ –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏, –∫–æ—Ç–æ—Ä—ã–µ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –≤ —è–∑—ã–∫–µ. –ü–æ-–ø—Ä–æ—Å—Ç–æ–º—É, —Ç–æ–∫–µ–Ω—ã –≤ —è–∑—ã–∫–µ ‚Äî —ç—Ç–æ –∫–∞–∫ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏ –ø—Ä–µ–ø–∏–Ω–∞–Ω–∏—è –≤ —Ç–µ–∫—Å—Ç–µ.

–ö–æ–º–ø—å—é—Ç–µ—Ä –ø–æ–Ω–∏–º–∞–µ—Ç —Ç–æ–ª—å–∫–æ —Å—Ç—Ä–æ–≥–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –ø–æ—Ä—è–¥–æ–∫ —Ç–æ–∫–µ–Ω–æ–≤. –°–∏–Ω—Ç–∞–∫—Å–∏—Å —è–∑—ã–∫–∞ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –≤ –∫–∞–∫–æ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –º–æ–≥—É—Ç —Å—Ç–æ—è—Ç—å —Ç–æ–∫–µ–Ω—ã, —á—Ç–æ–±—ã —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å –ø—Ä–∞–≤–∏–ª–∞–º –≥—Ä–∞–º–º–∞—Ç–∏–∫–∏ —è–∑—ã–∫–∞. –¢–æ–∫–µ–Ω—ã –Ω–µ –º–æ–≥—É—Ç —Å—Ç–æ—è—Ç—å –≤ –ª—é–±–æ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏, —Ç–∞–∫–∂–µ –∫–∞–∫ —Å–ª–æ–≤–∞ –≤ —ç—Ç–æ–º —Ç–µ–∫—Å—Ç–µ.

–ù–∞–ø—Ä–∏–º–µ—Ä, –≤ —è–∑—ã–∫–µ HDL –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —á–µ—Ç—ã—Ä–µ –∫–ª—é—á–µ–≤—ã—Ö —Å–ª–æ–≤–∞: `CHIP`, `IN`, `OUT`, `PARTS`. –ò –¥–≤–µ–Ω–∞–¥—Ü–∞—Ç—å –∑–Ω–∞–∫–æ–≤: `{ . ( : / [ } ] * = ) ,`. –í—Å–µ–≥–æ —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç—å —Ç–æ–∫–µ–Ω–æ–≤.

–ö–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏ –æ–±—Ä–∞–∑—É—é—Ç —Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫—É—é –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—é –æ–ø–∏—Å–∞–Ω–∏—è —ç–ª–µ–∫—Ç—Ä–æ—Å—Ö–µ–º—ã, –µ–µ HDL –∫–æ–¥ –Ω–∏–∂–µ.

```java
CHIP chip_name { // –Ω–∞–∑–≤–∞–Ω–∏–µ —Å—Ö–µ–º—ã

    OUT x, y, z; // –∞—É—Ç–ø—É—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å—Ö–µ–º–∞

    PARTS:
        // ‚Ä¶ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è –ª–æ–≥–∏—á–µ—Å–∫–æ–π —Å—Ö–µ–º—ã (—Å–∫—Ä—ã—Ç–∞) ‚Ä¶
}
```

–ó–µ–ª–µ–Ω—ã–º —Ü–≤–µ—Ç–æ–º –≤—ã–¥–µ–ª–µ–Ω—ã –∫–æ–º–º–µ–Ω—Ç–∞—Ä–∏–∏, –æ–Ω–∏ –Ω–µ —è–≤–ª—è—é—Ç—Å—è —á–∞—Å—Ç—å—é —Å—Ö–µ–º—ã.

–≠—Ç–∞ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤–æ –≤—Å–µ—Ö —Å—Ö–µ–º–∞—Ö (–≥–µ–π—Ç–∞—Ö –∏ —á–∏–ø–∞—Ö). –ö—Ä–æ–º–µ –Ω–µ–µ –≤ –¥–∞–Ω–Ω–æ–π —Ä–∞–∑–Ω–æ–≤–∏–¥–Ω–æ—Å—Ç–∏ HDL –Ω–∏—á–µ–≥–æ –Ω–µ—Ç, –ø–æ—ç—Ç–æ–º—É –≤—ã—É—á–∏—Ç—å —è–∑—ã–∫ –º–æ–∂–Ω–æ –∑–∞ –ø–æ–ª—á–∞—Å–∞ –ø—Ä–∞–∫—Ç–∏–∫–∏.

–ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ CHIP –æ–∑–Ω–∞—á–∞–µ—Ç –Ω–∞—á–∞–ª–æ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —á–∏–ø–∞. –ó–∞ –∫–ª—é—á–µ–≤—ã–º —Å–ª–æ–≤–æ–º –∏–¥–µ—Ç –Ω–∞–∑–≤–∞–Ω–∏–µ —á–∏–ø–∞. –í –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ —á–∏–ø –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è Not, —ç—Ç–æ —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã–π –≥–µ–π—Ç. –ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ CHIP –Ω–µ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —ç—Ç–æ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ —á–∏–ø, –æ–Ω–æ –æ–∑–Ω–∞—á–∞–µ—Ç –Ω–∞—á–∞–ª–æ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —ç–ª–µ–∫—Ç—Ä–æ—Å—Ö–µ–º—ã. –ê–≤—Ç–æ—Ä—ã —è–∑—ã–∫–∞ —Ä–µ—à–∏–ª–∏ –Ω–µ –≤–≤–æ–¥–∏—Ç—å –∫–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ GATE, —á—Ç–æ–±—ã –Ω–µ —É—Å–ª–æ–∂–Ω—è—Ç—å —Å–∏–Ω—Ç–∞–∫—Å–∏—Å —è–∑—ã–∫–∞. –ò –≥–µ–π—Ç—ã –∏ —á–∏–ø—ã –≤ —ç—Ç–æ–π —Ä–∞–∑–Ω–æ–≤–∏–¥–Ω–æ—Å—Ç–∏ —è–∑—ã–∫–æ–≤ HDL –Ω–∞—á–∏–Ω–∞—é—Ç—Å—è —Å–æ —Å–ª–æ–≤–∞ CHIP.

–ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ IN –æ–∑–Ω–∞—á–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –∏–Ω–ø—É—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç —Å—Ö–µ–º–∞. –ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ OUT –æ–∑–Ω–∞—á–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –∞—É—Ç–ø—É—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ —Å—Ö–µ–º–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç.

–ó–Ω–∞–∫–∏ `, { ; : }` —è–≤–ª—è—é—Ç—Å—è —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—è–º–∏. –ù–∞–ø—Ä–∏–º–µ—Ä —Ñ–∏–≥—É—Ä–Ω—ã–µ —Å–∫–æ–±–∫–∏ –æ–±–æ–∑–Ω–∞—á–∞—é—Ç { –Ω–∞—á–∞–ª–æ –∏ –∫–æ–Ω–µ—Ü } —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —á–∏–ø–∞. –ß–µ—Ä–µ–∑ –∑–∞–ø—è—Ç—É—é –ø–µ—Ä–µ—á–∏—Å–ª—è—é—Ç—Å—è –∏–Ω–ø—É—Ç—ã (a, b, c, ‚Ä¶) –∏ –∞—É—Ç–ø—É—Ç—ã (x, y, z, ‚Ä¶). –¢–æ—á–∫–∞ —Å –∑–∞–ø—è—Ç–æ–π –æ–∑–Ω–∞—á–∞–µ—Ç –∫–æ–Ω–µ—Ü –æ–±—ä—è–≤–ª–µ–Ω–∏—è –∏–Ω–ø—É—Ç–æ–≤ –∏ –∞—É—Ç–ø—É—Ç–æ–≤.

–ò–Ω–ø—É—Ç—ã –∏ –∞—É—Ç–ø—É—Ç—ã ‚Äî —ç—Ç–æ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å —á–∏–ø–∞. –ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ PARTS –∏ –∑–Ω–∞–∫ : (PARTS:) –æ–∑–Ω–∞—á–∞—é—Ç –Ω–∞—á–∞–ª–æ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏.

–î–∞–≤–∞–π—Ç–µ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –æ–ø–∏—Å–∞–Ω–∏—è —Ä–∞–∑–Ω—ã—Ö –≥–µ–π—Ç–æ–≤ –Ω–∞ HDL.

–ù–∞—á–Ω–µ–º —Å –≥–µ–π—Ç–∞ NOT.

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º –µ—â–µ –æ–¥–∏–Ω –ø—Ä–∏–º–µ—Ä:

```java
        // ‚Ä¶ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è –ª–æ–≥–∏—á–µ—Å–∫–æ–π —Å—Ö–µ–º—ã (–ø–æ–∫–∞ —Å–∫—Ä—ã—Ç–∞) ‚Ä¶
```

–ö–∞–∫ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –≥–µ–π—Ç? –°–∫–æ–ª—å–∫–æ –∏–Ω–ø—É—Ç–æ–≤ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç? –°–∫–æ–ª—å–∫–æ –¥–æ–ª–∂–µ–Ω –≤–µ—Ä–Ω—É—Ç—å –∞—É—Ç–ø—É—Ç–æ–≤?

–û—Ç–≤–µ—Ç—ã (–Ω–µ —á–∏—Ç–∞–π—Ç–µ —Å—Ä–∞–∑—É, –ø–æ–¥—É–º–∞–π—Ç–µ): –ß–∏–ø –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è And, –ø–æ–ª—É—á–∞–µ—Ç –¥–≤–∞ –∏–Ω–ø—É—Ç–∞ `a`, `b` –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –æ–¥–∏–Ω –∞—É—Ç–ø—É—Ç `out`.

–¢–µ–ø–µ—Ä—å, –∫–æ–≥–¥–∞ –º—ã —Ä–∞–∑–æ–±—Ä–∞–ª–∏—Å—å, —á—Ç–æ –æ–∑–Ω–∞—á–∞—é—Ç –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏, –¥–∞–≤–∞–π—Ç–µ –ø–µ—Ä–µ–π–¥–µ–º –∫ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö —Å—Ö–µ–º.

## –†–µ–∞–ª–∏–∑–∞—Ü–∏—è –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö —Å—Ö–µ–º –Ω–∞ HDL

–î–ª—è –ø—Ä–∏–º–µ—Ä–∞ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—é —Å–æ—Å—Ç–∞–≤–Ω–æ–≥–æ –≥–µ–π—Ç–∞ NAND –∏–∑ –≥–µ–π—Ç–æ–≤ AND –∏ NOT.

```java
CHIP Nand {

    IN x, y; // –ø–æ–ª—É—á–∞–µ–º –¥–≤–∞ –∏–Ω–ø—É—Ç–∞: x, y;
    OUT z; // –¥–æ–ª–∂–Ω—ã –≤–µ—Ä–Ω—É—Ç—å –æ–¥–∏–Ω –∞—É—Ç–ø—É—Ç: z;

    PARTS:
    // –≤ –≥–µ–π—Ç And –ø–µ—Ä–µ–¥–∞–µ–º –æ–±–∞ –≤—Ö–æ–¥—è—â–∏—Ö –∏–Ω–ø—É—Ç–∞: a=x, b=y;
    // out –∑–∞–ø–∏—Å—ã–≤–∞–µ–º –≤ xANDy –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é (–ø–∏–Ω);
    And(a=x, b=y, out=xANDy);
    // –¥–∞–ª–µ–µ –ø–µ—Ä–µ–¥–∞–µ–º –ø–∏–Ω xANDy –≤ –∏–Ω–ø—É—Ç –≥–µ–π—Ç–∞ Not: in=xANDy;
    // –ø–æ–ª—É—á–µ–Ω–Ω—ã–π out –≤—ã–≤–æ–¥–∏–º –≤ –≥–ª–∞–≤–Ω—ã–π out: out=z;
    Not(in=xANDy, out=z);

}
```

–¢–µ–ø–µ—Ä—å –ø–æ—Å–º–æ—Ç—Ä–∏–º –Ω–∞ —Å–æ—Å—Ç–∞–≤–Ω–æ–π –≥–µ–π—Ç Nand –µ—â–µ —Ä–∞–∑ –±–µ–∑ –∑–µ–ª–µ–Ω—ã—Ö –∫–æ–º–º–µ–Ω—Ç–∞—Ä–∏–µ–≤.

```java
CHIP Nand {
```
