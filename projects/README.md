# HDL (—è–∑—ã–∫ –æ–ø–∏—Å–∞–Ω–∏—è –∂–µ–ª–µ–∑–∞ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞)

HDL (hardware description language) ‚Äî —ç—Ç–æ —è–∑—ã–∫ –Ω–∞ –∫–æ—Ç–æ—Ä–æ–º –ø—Ä–æ–µ–∫—Ç–∏—Ä—É—é—Ç –∞–ø–ø–∞—Ä–∞—Ç–Ω–æ–µ –æ–±–µ—Å–ø–µ—á–µ–Ω–∏–µ (–∂–µ–ª–µ–∑–æ) –∫–æ–º–ø—å—é—Ç–µ—Ä–∞. HDL –∫–æ–¥ –∑–∞–ø—É—Å–∫–∞—é—Ç –≤ —Å–∏–º—É–ª—è—Ç–æ—Ä–µ –∂–µ–ª–µ–∑–∞. –°–∏–º—É–ª—è—Ç–æ—Ä –∂–µ–ª–µ–∑–∞ ‚Äî —ç—Ç–æ –ø—Ä–æ–≥—Ä–∞–º–º–∞, –∫–æ—Ç–æ—Ä–∞—è —Å–∏–º—É–ª–∏—Ä—É–µ—Ç –∫–æ–º–ø—å—é—Ç–µ—Ä–Ω—É—é —Å–∏—Å—Ç–µ–º—É.

–í XXI –≤–µ–∫–µ —á–∏–ø—ã –Ω–µ –ø–∞—è—é—Ç —Ä—É–∫–∞–º–∏, –∏—Ö –ø—Ä–æ–∏–∑–≤–æ–¥—è—Ç –Ω–∞ –∑–∞–≤–æ–¥–∞—Ö. –í—ã –ø–∏—à–∏—Ç–µ HDL –∫–æ–¥ ‚Äî —Ä–æ–±–æ—Ç-–ø–∞—è–ª—å–Ω–∏–∫ –Ω–∞ –∑–∞–≤–æ–¥–µ –ø–∞—è–µ—Ç —á–∏–ø. –ü—Ä–æ–µ–∫—Ç–∏—Ä–æ–≤–∞—Ç—å —á–∏–ø –≤ —Å–∏–º—É–ª—è—Ç–æ—Ä–µ –∂–µ–ª–µ–∑–∞ –ø—Ä–æ—â–µ, —á–µ–º –ø–∞—è—Ç—å –µ–≥–æ —Ä—É–∫–∞–º–∏. –¶–µ–Ω–∞ –æ—à–∏–±–∫–∏ –º–µ–Ω—å—à–µ, –ø—Ä–æ—Ü–µ—Å—Å —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–∏ –±—ã—Å—Ç—Ä–µ–µ.

–î–≤–∞ –æ—Å–Ω–æ–≤–Ω—ã—Ö HDL —è–∑—ã–∫–∞ ‚Äî System Verilog –∏ VHDL. –ò–º–µ–Ω–Ω–æ –∏—Ö –∏—Å–ø–æ–ª—å–∑—É—é—Ç –≤ –ø—Ä–æ–¥–∞–∫—à–µ–Ω–µ, –Ω–æ –º—ã –Ω–µ –±—É–¥–µ–º –∏—Ö —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å, –ø–æ—Ç–æ–º—É —á—Ç–æ –æ–Ω–∏ —Å–ª–∏—à–∫–æ–º –±–æ–ª—å—à–∏–µ. –ú—ã –±—É–¥–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—É—é —É–ø—Ä–æ—â–µ–Ω–Ω—É—é –≤–µ—Ä—Å–∏—é HDL, –ø–æ—Ö–æ–∂—É—é –Ω–∞ System Verilog.

## –°–∏–Ω—Ç–∞–∫—Å–∏—Å —è–∑—ã–∫–∞

–õ—é–±–æ–π —è–∑—ã–∫ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ —Ç–æ–∫–µ–Ω–æ–≤, HDL –Ω–µ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ. –¢–æ–∫–µ–Ω—ã ‚Äî —ç—Ç–æ –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏, –∫–æ—Ç–æ—Ä—ã–µ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –≤ —è–∑—ã–∫–µ. –ü–æ-–ø—Ä–æ—Å—Ç–æ–º—É, —Ç–æ–∫–µ–Ω—ã –≤ —è–∑—ã–∫–µ ‚Äî —ç—Ç–æ –∫–∞–∫ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏ –ø—Ä–µ–ø–∏–Ω–∞–Ω–∏—è –≤ —Ç–µ–∫—Å—Ç–µ.

–¢–æ–∫–µ–Ω—ã –Ω–µ –º–æ–≥—É—Ç —Å—Ç–æ—è—Ç—å –≤ –ª—é–±–æ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏, —Ç–∞–∫–∂–µ –∫–∞–∫ —Å–ª–æ–≤–∞ –≤ —Ç–µ–∫—Å—Ç–µ. –ö–æ–º–ø—å—é—Ç–µ—Ä –ø–æ–Ω–∏–º–∞–µ—Ç —Ç–æ–ª—å–∫–æ —Å—Ç—Ä–æ–≥–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –ø–æ—Ä—è–¥–æ–∫ —Ç–æ–∫–µ–Ω–æ–≤. –≠—Ç–æ—Ç –ø–æ—Ä—è–¥–æ–∫ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Å–∏–Ω—Ç–∞–∫—Å–∏—Å.

–ù–∞–ø—Ä–∏–º–µ—Ä, –≤ —è–∑—ã–∫–µ HDL –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —á–µ—Ç—ã—Ä–µ –∫–ª—é—á–µ–≤—ã—Ö —Å–ª–æ–≤–∞: `CHIP`, `IN`, `OUT`, `PARTS`. –ò –¥–≤–µ–Ω–∞–¥—Ü–∞—Ç—å –∑–Ω–∞–∫–æ–≤: `{ . ( : / [ } ] * = ) ,`. –í—Å–µ–≥–æ —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç—å —Ç–æ–∫–µ–Ω–æ–≤. –ù–µ —Ç–∞–∫ –º–Ω–æ–≥–æ, –≤–µ—Å—å —è–∑—ã–∫ –º–æ–∂–Ω–æ –≤—ã—É—á–∏—Ç—å –∑–∞ –ø–æ–ª—á–∞—Å–∞.

–ö–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏ –æ–±—Ä–∞–∑—É—é—Ç —Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫—É—é –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—é –æ–ø–∏—Å–∞–Ω–∏—è —ç–ª–µ–∫—Ç—Ä–æ—Å—Ö–µ–º—ã. –≠—Ç–∞ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤–æ –≤—Å–µ—Ö —Å—Ö–µ–º–∞—Ö, –µ–µ HDL –∫–æ–¥ –Ω–∏–∂–µ:

```java
CHIP chip_name { // –Ω–∞–∑–≤–∞–Ω–∏–µ —Å—Ö–µ–º—ã
    IN a, b, c; // –∏–Ω–ø—É—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç —Å—Ö–µ–º–∞
    OUT x, y, z; // –∞—É—Ç–ø—É—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å—Ö–µ–º–∞

    PARTS:
        // ‚Ä¶ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è –ª–æ–≥–∏—á–µ—Å–∫–æ–π —Å—Ö–µ–º—ã (—Å–∫—Ä—ã—Ç–∞) ‚Ä¶
}
```

–°–ª–µ—à–∞–º–∏ `//` –≤—ã–¥–µ–ª–µ–Ω—ã –∫–æ–º–º–µ–Ω—Ç–∞—Ä–∏–∏, –æ–Ω–∏ –Ω–µ —è–≤–ª—è—é—Ç—Å—è —á–∞—Å—Ç—å—é —Å—Ö–µ–º—ã.

–ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `CHIP` –æ–∑–Ω–∞—á–∞–µ—Ç –Ω–∞—á–∞–ª–æ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —á–∏–ø–∞. –ó–∞ –∫–ª—é—á–µ–≤—ã–º —Å–ª–æ–≤–æ–º –∏–¥–µ—Ç –Ω–∞–∑–≤–∞–Ω–∏–µ —á–∏–ø–∞. –í –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ —á–∏–ø –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è `chip_name`, —ç—Ç–æ –ø—Ä–∏–º–µ—Ä –Ω–∞–∑–≤–∞–Ω–∏—è. –ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `CHIP` –Ω–µ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —ç–ª–µ–∫—Ç—Ä–æ—Å—Ö–µ–º–∞ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ —á–∏–ø, –æ–Ω–æ –æ–∑–Ω–∞—á–∞–µ—Ç –Ω–∞—á–∞–ª–æ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —ç–ª–µ–∫—Ç—Ä–æ—Å—Ö–µ–º—ã. –ê–≤—Ç–æ—Ä—ã —è–∑—ã–∫–∞ —Ä–µ—à–∏–ª–∏ –Ω–µ –≤–≤–æ–¥–∏—Ç—å –∫–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `GATE`, —á—Ç–æ–±—ã –Ω–µ —É—Å–ª–æ–∂–Ω—è—Ç—å —è–∑—ã–∫. –ò –≥–µ–π—Ç—ã –∏ —á–∏–ø—ã –≤ –Ω–∞—à–µ–º HDL –Ω–∞—á–∏–Ω–∞—é—Ç—Å—è —Å–æ —Å–ª–æ–≤–∞ `CHIP`.

–ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `IN` –æ–∑–Ω–∞—á–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –∏–Ω–ø—É—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç —Å—Ö–µ–º–∞. –ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `OUT` –æ–∑–Ω–∞—á–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –∞—É—Ç–ø—É—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ —Å—Ö–µ–º–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç.

–ó–Ω–∞–∫–∏ `, { ; : }` —è–≤–ª—è—é—Ç—Å—è —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—è–º–∏. –ù–∞–ø—Ä–∏–º–µ—Ä —Ñ–∏–≥—É—Ä–Ω—ã–µ —Å–∫–æ–±–∫–∏ –æ–±–æ–∑–Ω–∞—á–∞—é—Ç `{ –Ω–∞—á–∞–ª–æ –∏ –∫–æ–Ω–µ—Ü }` —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —Å—Ö–µ–º—ã. –ß–µ—Ä–µ–∑ –∑–∞–ø—è—Ç—É—é –ø–µ—Ä–µ—á–∏—Å–ª—è—é—Ç—Å—è –∏–Ω–ø—É—Ç—ã `IN: a, b, c` –∏ –∞—É—Ç–ø—É—Ç—ã `OUT: x, y, z`. –¢–æ—á–∫–∞ —Å –∑–∞–ø—è—Ç–æ–π –æ–∑–Ω–∞—á–∞–µ—Ç –∫–æ–Ω–µ—Ü –æ–±—ä—è–≤–ª–µ–Ω–∏—è –∏–Ω–ø—É—Ç–æ–≤ –∏ –∞—É—Ç–ø—É—Ç–æ–≤.

–ò–Ω–ø—É—Ç—ã –∏ –∞—É—Ç–ø—É—Ç—ã ‚Äî —ç—Ç–æ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å —Å—Ö–µ–º—ã. –ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `PARTS` –æ–∑–Ω–∞—á–∞–µ—Ç –Ω–∞—á–∞–ª–æ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —Å—Ö–µ–º—ã.

–î–∞–≤–∞–π—Ç–µ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –æ–ø–∏—Å–∞–Ω–∏—è —Ä–∞–∑–Ω—ã—Ö –≥–µ–π—Ç–æ–≤ –Ω–∞ HDL. –ù–∞—á–Ω–µ–º —Å –≥–µ–π—Ç–∞ NOT:

```java
CHIP Not { // –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è Not
    IN in; // –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –æ–¥–∏–Ω –∏–Ω–ø—É—Ç: in;
    OUT out; // –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –æ–¥–∏–Ω –∞—É—Ç–ø—É—Ç: out;

    PARTS:
        // ‚Ä¶ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è –ª–æ–≥–∏—á–µ—Å–∫–æ–π —Å—Ö–µ–º—ã NOT (–ø–æ–∫–∞ —Å–∫—Ä—ã—Ç–∞) ‚Ä¶
}
```

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º –µ—â–µ –æ–¥–∏–Ω –ø—Ä–∏–º–µ—Ä:

```java
CHIP And {
    IN a, b;
    OUT out;

    PARTS:
        // ‚Ä¶ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è –ª–æ–≥–∏—á–µ—Å–∫–æ–π —Å—Ö–µ–º—ã (–ø–æ–∫–∞ —Å–∫—Ä—ã—Ç–∞) ‚Ä¶

}
```

–ö–∞–∫ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –≥–µ–π—Ç? –°–∫–æ–ª—å–∫–æ –∏–Ω–ø—É—Ç–æ–≤ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç? –°–∫–æ–ª—å–∫–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∞—É—Ç–ø—É—Ç–æ–≤?

–û—Ç–≤–µ—Ç—ã (–Ω–µ —á–∏—Ç–∞–π—Ç–µ —Å—Ä–∞–∑—É, –ø–æ–¥—É–º–∞–π—Ç–µ): –ß–∏–ø –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è `And`, –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –¥–≤–∞ –∏–Ω–ø—É—Ç–∞ `a`, `b` –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –æ–¥–∏–Ω –∞—É—Ç–ø—É—Ç `out`.

–¢–µ–ø–µ—Ä—å, –∫–æ–≥–¥–∞ –º—ã —Ä–∞–∑–æ–±—Ä–∞–ª–∏—Å—å, —á—Ç–æ –æ–∑–Ω–∞—á–∞—é—Ç –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏, –¥–∞–≤–∞–π—Ç–µ –ø–µ—Ä–µ–π–¥–µ–º –∫ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö —Å—Ö–µ–º.

## –†–µ–∞–ª–∏–∑–∞—Ü–∏—è –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö —Å—Ö–µ–º –Ω–∞ HDL

–î–ª—è –ø—Ä–∏–º–µ—Ä–∞ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—é —Å–æ—Å—Ç–∞–≤–Ω–æ–≥–æ –≥–µ–π—Ç–∞ NAND –∏–∑ –≥–µ–π—Ç–æ–≤ AND –∏ NOT.

```java
CHIP Nand {

    IN x, y; // –ø–æ–ª—É—á–∞–µ–º –¥–≤–∞ –∏–Ω–ø—É—Ç–∞: x, y;
    OUT z; // –¥–æ–ª–∂–Ω—ã –≤–µ—Ä–Ω—É—Ç—å –æ–¥–∏–Ω –∞—É—Ç–ø—É—Ç: z;

    PARTS:
    // –≤ –≥–µ–π—Ç And –ø–µ—Ä–µ–¥–∞–µ–º –æ–±–∞ –≤—Ö–æ–¥—è—â–∏—Ö –∏–Ω–ø—É—Ç–∞: a<-x, b<-y;
    // out –∑–∞–ø–∏—Å—ã–≤–∞–µ–º –≤ xANDy —à–∏–Ω—É: out->xANDy;
    And(a=x, b=y, out=xANDy);
    // –¥–∞–ª–µ–µ –ø–µ—Ä–µ–¥–∞–µ–º —à–∏–Ω—É xANDy –≤ –∏–Ω–ø—É—Ç –≥–µ–π—Ç–∞ Not: in<-xANDy;
    // –ø–æ–ª—É—á–µ–Ω–Ω—ã–π out –≤—ã–≤–æ–¥–∏–º –≤ –≥–ª–∞–≤–Ω—ã–π out: out->z;
    Not(in=xANDy, out=z);

}
```

–ì–µ–π—Ç NAND –ø–æ–ª—É—á–∞–µ—Ç –¥–≤–∞ –∏–Ω–ø—É—Ç–∞: `x` –∏ `y`. –ü–µ—Ä–µ–¥–∞–µ–º –∏—Ö –≤ –≥–µ–π—Ç And, —Ä–µ–∑—É–ª—å—Ç–∞—Ç And –æ–ø–µ—Ä–∞—Ü–∏–∏ –∑–∞–ø–∏—Å—ã–≤–∞–µ–º –≤ —à–∏–Ω—É `xANDy`: `And(a=x, b=y, out=xANDy);`. –ó–¥–µ—Å—å —à–∏–Ω–∞ `xANDy` –∏–≥—Ä–∞–µ—Ç —Ä–æ–ª—å –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –µ–µ –º–æ–∂–Ω–æ –Ω–∞–∑–≤–∞—Ç—å –∫–∞–∫ —É–¥–æ–±–Ω–æ, —è –Ω–∞–∑–≤–∞–ª `xANDy`. –î–∞–ª–µ–µ –ø–µ—Ä–µ–¥–∞–µ–º —à–∏–Ω—É `xANDy` –≤ –≥–µ–π—Ç Not –∏ –∑–∞–ø–∏—Å—ã–≤–∞–µ–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç –æ–ø–µ—Ä–∞—Ü–∏–∏ Not –≤ `z`. –ê—É—Ç–ø—É—Ç `z` —Å—á–∏—Ç–∞–µ—Ç—Å—è –æ—Å–Ω–æ–≤–Ω—ã–º, —Å–∏–º—É–ª—è—Ç–æ—Ä –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –ø–æ–¥—Ö–≤–∞—Ç–∏—Ç –µ–≥–æ, –∫–æ–≥–¥–∞ –≤—ã —Å–æ–∑–¥–∞–¥–∏—Ç–µ —à–∏–Ω—É —Å –Ω–∞–∑–≤–∞–Ω–∏–µ–º `z`.

–¢–µ–ø–µ—Ä—å –ø–æ—Å–º–æ—Ç—Ä–∏–º –Ω–∞ —Å–æ—Å—Ç–∞–≤–Ω–æ–π –≥–µ–π—Ç Nand –µ—â–µ —Ä–∞–∑ –±–µ–∑ –∫–æ–º–º–µ–Ω—Ç–∞—Ä–∏–µ–≤.

```java
CHIP Nand {
    IN x, y;
    OUT z;

    PARTS:
        And(a=x, b=y, out=xANDy);
        Not(in=xANDy, out=z);
```
